# 🌙 أكاديمية سحر قمر (Sahar Qamar Academy)

منصة تعليم إلكتروني متكاملة — واجهة أمامية ساحرة (HTML/CSS/JS) + باك اند PHP/MySQL آمن + لوحة تحكم كاملة.

## 📁 محتويات المشروع
```
sahar-qamar-academy/
├── index.html, about.html, courses.html, details.html, contact.html, login.html, register.html
├── style.css     ← الهوية اللونية (بنفسجي ليلي + توهج قمري فضي + ذهبي) والحركات
├── script.js      ← منطق الواجهة + الربط مع الـ API + حركات الظهور عند التمرير
├── lang.js         ← نظام تبديل اللغة (عربي/إنجليزي)
└── backend/
    ├── .htaccess       ← حماية ملفات الإعداد وقاعدة البيانات
    ├── database.sql     ← قاعدة البيانات (12 دورة تجريبية جاهزة)
    ├── config.php        ← إعدادات الاتصال بقاعدة البيانات
    ├── includes/helpers.php
    ├── api/                ← واجهات JSON (تسجيل، دخول، دورات، تعليقات، تواصل) — محمية ببروتوكول prepared statements
    └── admin/                ← لوحة التحكم الكاملة (محمية بـ CSRF + قفل محاولات الدخول)
```

## 🚀 التشغيل
1. ثبّت XAMPP (أو أي سيرفر PHP + MySQL محلي).
2. انسخ مجلد `sahar-qamar-academy` بالكامل داخل `htdocs`.
3. من phpMyAdmin استورد ملف `backend/database.sql` (سينشئ قاعدة البيانات تلقائياً).
4. تأكد من `backend/config.php` (الإعداد الافتراضي مطابق لـ XAMPP: مستخدم `root` بدون كلمة مرور).
5. افتح الموقع: `http://localhost/sahar-qamar-academy/index.html`
6. لوحة التحكم: `http://localhost/sahar-qamar-academy/backend/admin/login.php`

> الواجهة الأمامية تعمل حتى بدون تشغيل سيرفر (بفتح index.html مباشرة) بفضل بيانات تجريبية احتياطية، لكن الحفظ الحقيقي (تسجيل، تعليقات، رسائل) يتطلب تشغيل المشروع عبر سيرفر PHP.

## ✨ أبرز التحديثات في هذه النسخة
- **الاسم والهوية:** أكاديمية سحر قمر 🌙 بألوان بنفسجي ليلي ساحر + توهج قمري فضي + ذهبي، مع نجوم متلألئة وحركة قمر عائم في البانر الرئيسي.
- **قائمة همبرجر** تعمل الآن في كل أحجام الشاشات (جوال وحاسوب) بقائمة منسدلة أنيقة.
- **12 دورة تدريبية** (بدلاً من 6) موزعة على 4 تصنيفات، بتفاصيل أوسع لكل دورة.
- **صفحة "من نحن" موسّعة**: قصة الأكاديمية، لماذا نحن مختلفون، إحصائيات، رؤية ورسالة وأهداف، فريق العمل.
- **حركات ظهور تدريجي (Scroll Reveal)** عند التمرير في كل الصفحات لتجربة أكثر حيوية.
- **توسيط العناوين الرئيسية** في صفحات الرئيسية/الدورات/من نحن/تواصل لعرض أوضح على كل الشاشات.
- **تحسينات أمان حقيقية:**
  - تشفير كلمات المرور بخوارزمية bcrypt.
  - استعلامات محضّرة (Prepared Statements) في كل نقاط الاتصال بقاعدة البيانات لمنع SQL Injection.
  - حماية CSRF لنماذج لوحة التحكم.
  - تجديد معرف الجلسة (Session Regeneration) بعد كل تسجيل دخول ناجح.
  - قفل مؤقت بعد محاولات دخول فاشلة متكررة (لوحة التحكم وتسجيل دخول الطلاب).
  - ملف `.htaccess` يمنع الوصول المباشر لملفات الإعداد وقاعدة البيانات ويعطّل عرض محتويات المجلدات.
  - رؤوس أمان HTTP إضافية (X-Frame-Options, X-Content-Type-Options).
- **استجابة كاملة لكل الشاشات**: من أصغر جوال إلى أكبر شاشة حاسوب، بدون أي كسر بالتصميم.
